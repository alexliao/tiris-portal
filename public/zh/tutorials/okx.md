# 在 OKX 创建仅供 Tiris 使用的 API Key

本向导仅供 Tiris 绑定：只需读取资产与现货下单/撤单权限，不含提现、划转、期货/杠杆等能力。

## 前置准备
- OKX 账户已完成 KYC，并开启手机/邮箱 + 2FA 验证。
- 建议使用桌面网页端操作，避免移动端界面差异。

## 创建与配置 API Key
1) 登录后，点击右上角头像 → 选择「API」。直达链接：https://okx.com/zh-hans/account/my-api  
2) 点击「创建 API Key」。  
3) 填写「备注名」（如 “Tiris专用”）
4) 「账户」：如果不确定，就选择「主账户」。
5) 「IP 地址白名单」：可先不限制，后续按需补充。  
6) 「权限」：  
   - 选择「读取」和「交易」权限。  
   - **不要** 选择「提现」。  
7) 设置「密码（Passphrase）」，请妥善保存。     
8) 确认信息后点击「提交全部」，按提示完成短信/邮箱/谷歌验证。系统会生成 `API Key`、`Secret Key` 以及刚设置的 `Passphrase`。  
9) 立即复制并安全保存上述三项信息，密钥通常只显示一次。

## 在 Tiris 中使用
1) 打开 Tiris → 进入交易所绑定向导 → 选择 OKX。  
2) 将 `API Key`、`Secret Key`、`Passphrase` 粘贴到对应输入框。  
3) 确认权限仅为「读取 + 交易」，完成绑定。

## 常见检查
- 绑定失败：核对是否选择了「读取 + 交易」权限，或误勾选了「提款」。  
- 如开启 IP 白名单，确认已加入 Tiris 访问 IP（172.105.23.166）。  
- 如果需要更换密钥，在 OKX 删除旧密钥后重新创建，再在 Tiris 中更新。
