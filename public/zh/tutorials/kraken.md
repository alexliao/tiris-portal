# 在 Kraken 创建仅供 Tiris 使用的 API Key

本向导仅供 Tiris 绑定：只需读取资产与现货下单/撤单权限，不含提现、划转、杠杆/期货等能力。

## 前置准备
- Kraken 账户已通过 KYC，并开启 2FA（Google Authenticator 等）。
- 建议在桌面网页端操作，避免移动端界面差异。

## 创建与配置 API Key
1) 登录Kraken pro，点击右上角头像 → 选择「设置」→ 「连接和API」。直达链接：https://pro.kraken.com/app/settings/api  
2) 点击「创建 API 密钥」。  
3) 设置一个名称（如 “Tiris专用”）。  
4) 权限最小化，仅勾选：  
   - `资金权限` - `查询`  
   - `订单和交易` - 全部子选项
   - 其他权限全部保持不选，尤其是**不要**选 `提款`。  
5) 其它选项保持「关闭」。
7) 点击「生成密钥」，按提示完成 2FA验证。页面会生成 `API Key` 与 `API secret`（仅显示一次）。  
8) 立即复制并安全保存上述两项信息。

## 在 Tiris 中使用
1) 打开 Tiris → 进入交易所绑定向导 → 选择 Kraken。  
2) 将 `API Key` 与 `Private Key` 粘贴到对应输入框。  
3) 确认权限仅为「读取 + 现货交易」，完成绑定。

## 常见检查
- 绑定失败：核对是否勾选了 `资金权限` - `查询`，或误勾选提现/转账/其他高权限。  
- 如启用 IP 白名单，确认已加入 172.105.23.166。  
- 需要更换密钥时，先在 Kraken 删除旧密钥，再创建新密钥并在 Tiris 更新。
