# 在 Gate 创建仅供 Tiris 使用的 API Key

以下步骤只为 Tiris 绑定向导准备：权限最小化，仅允许读取现货账户信息与交易，不含提币/划转/合约等能力。

## 前置准备
- 拥有已完成 KYC 的 Gate 账户，并开启谷歌验证/短信等二次验证。
- 建议使用电脑网页端完成操作，避免移动端界面差异。

## 创建与配置 API Key
1) 登录后，点击右上角头像 → 选择「API管理」。直达链接：https://www.gate.com/zh/myaccount/profile/api-key/manage
2) 点击「创建 API Key」。  
3) 填写备注，便于识别（如 “Tiris专用”）。  
4) IP 白名单：可先选择不填，以后再补充。  
5) 权限勾选（保持最小化）：  
   - 勾选「现货」。  
   - 勾选「读写」。  
   - **不要** 勾选「提现」「资金划转」「合约/期货读取或交易」「杠杆借贷」等其他权限。  
6) 点击「提交」创建，按提示完成二次验证。Gate 会生成 `API Key` 与 `API Secret`（密钥只显示一次）。  
7) 立即复制并安全保存这两个值，不要在公开场合泄露。

## 在 Tiris 中使用
1) 打开 Tiris → 进入交易所绑定向导 → 选择 Gate。  
2) 将刚获得的 `API Key` 与 `Secret Key` 粘贴到对应输入框。  
3) 确认权限仅为「读取 + 现货交易」，完成绑定。

## 常见检查
- 绑定失败时，先核对：是否遗漏「现货」勾选，或误勾选了提现/合约权限。  
- 如开启了 IP 白名单，确认 Tiris 访问 IP（172.105.23.166）已加入。  
- 若要更换密钥，需要在 Gate 删除旧密钥并重新创建，再在 Tiris 中更新。
