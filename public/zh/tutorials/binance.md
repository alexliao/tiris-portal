# 在 Binance 创建仅供 Tiris 使用的 API Key

本向导仅供 Tiris 绑定：只需读取资产与现货下单/撤单权限，不含提现、划转、合约/杠杆等能力。

## 前置准备
- Binance 账户已完成 KYC，并开启谷歌验证器/短信等二次验证。
- 建议使用桌面网页端操作，避免移动端界面差异。

## 创建与配置 API Key
1) 登录后，点击右上角头像, 选择`账户` → `API 管理`。直达链接：https://www.binance.com/zh-CN/my/settings/api-management
2) 点击`创建 API`。
3) 选择`系统生成`类型的 API Key，点击`下一步`。
4) 输入API标签（如 "Tiris专用"），
5) 按提示完成二次验证（邮箱 + 谷歌验证器/短信）。
6) 创建成功后，系统会生成 `API密钥` 与 `密钥`（密钥只显示一次）。
7) 立即复制并安全保存上述两项信息。
8) 点击`编辑权限`配置权限：
   - `IP 访问限制`：选择`只访问受信任的IP(推荐)`，输入Tiris的访问IP “172.105.23.166”，点`确认`
   - 勾选`允许读取`和`允许现货及杠杆交易`。
   - **不要**勾选`允许提现`
9) 点击`保存`，完成二次验证确认。

## 在 Tiris 中使用
1) 打开 Tiris → 进入交易所绑定向导 → 选择 `Binance`。
2) 将 `API密钥` 与 `密钥` 粘贴到`API Key`和`API Secret`输入框。
3) 点`验证凭证`验证成功后，点`下一步`完成绑定。

## 常见检查
- 绑定失败：核对是否勾选了`允许现货及杠杆交易`，或误勾选了`允许提现`。
- 如开启了 `IP 访问限制`，确认已加入 Tiris 访问 IP（172.105.23.166）。
- 如果需要更换密钥，在 Binance 删除旧密钥后重新创建，再在 Tiris 中更新。
